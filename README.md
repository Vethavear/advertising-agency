## Project Overview 🎉
Simple food searcher using https://world.openfoodfacts.org api. Project done to practice Axios and MVC. Suitable for mobile devices, although only in landscape orientation.
## Tech/framework used 🔧

| Tech                                                    
| -------------------------------------------------------
| HTML                           
| SASS                           
| JS                           
| Axios
